Allows you to apply watermark on woocommerce products images

== Description ==

Product Image Watermark for Woo plugin allows you to apply watermark on WooCommerce products images which should be a downloadable products. These watermarks can be an image copyright symbol, company logo or piece of branding text as a transparent PNG image.

= Features: =
&#9989; Automatically add watermark for WooCommerce downloadable product images.
&#9989; You can also apply watermarks to an already uploaded product images. 
&#9989; Repeat watermark on a single image
&#9989; Watermark on existing product images
&#9989; Align a Watermark to the specific position like in middle, middle left, middle right, top left, top center, top right, bottom left, bottom center & bottom right.
&#9989; Apply Watermark on any size of an image
&#9989; User friendly admin UI



== Installation ==
Important: First of all, WooCommerce plugin must be activated on your wordpress website, without it Product Image Watermark for Woocommerce plugin will not work.

1. Unzip the downloaded .zip file.
2. Upload the Product Image Watermark for Woo plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `Product Image Watermark for Woo` from Plugins menu

= Configuration =
Product Image Watermark for Woo will add a new setting menu to the WooCommerce setting page with setting name "Download Image Watermark". There you will be able to set the watermark image for each image sizes and set the position of watermark image.


== Frequently Asked Questions ==

= Why watermark not applied to some images? I have already regenerated thumbnails. =
Please make sure you have attached image to any product from the admin menu Media -> Library
